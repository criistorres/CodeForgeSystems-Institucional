Utilidades

Algumas utilidades estão disponíveis por padrão em qualquer template Fluid do JJInfinity.

Funções Default

1. localize

Localiza uma string com base em uma chave de recurso e parâmetros opcionais.

{{ localize("ChaveDeRecurso", "param1", "param2") }}

A função recupera a string localizada do arquivo de recursos MasterDataResources usando a chave fornecida, caso não seja encontrado nenhum valor, será buscado no cadastro de traduções. Parâmetros opcionais podem ser passados para formatação de string.

2. formatDate

Formata um objeto DateTime em uma string de acordo com um formato especificado.

{{ formatDate(objData, "dd/MM/yyyy") }}

Se o primeiro argumento for um DateTime válido, ele retorna uma string de data formatada. Caso contrário, retorna a representação padrão do ToString() do objeto.

3. isNullOrEmpty

Verifica se a string fornecida é null ou vazia.

{{ isNullOrEmpty(minhaString) }}

Retorna true se a string for null ou vazia, false caso contrário.

4. isNullOrWhiteSpace

Verifica se a string fornecida é null, vazia ou contém apenas caracteres de espaço em branco.

{{ isNullOrWhiteSpace(minhaString) }}

Retorna true se a string for null, vazia ou composta apenas por espaços em branco, false caso contrário.

5. substring

Extrai uma substring de uma string fornecida, começando no índice especificado e com um comprimento opcional.

{{ substring(minhaString, 0, 5) }}

Se apenas o índice inicial for fornecido, a função retorna a substring a partir desse índice até o final da string. Se ambos o índice inicial e o comprimento forem fornecidos, ela extrai aquele intervalo de caracteres.

6. urlPath

Retorna o caminho da URL da aplicação.

<div class="avatar avatar-m">
    <img src="{{urlPath()}}Identidade/Perfil/AvatarPorIdLegado?legacyId={{CodUser}}" style="width:30px" class="rounded-circle">
</div>

6. dateAsText

Retorna a data como uma representação em string.

{{ dateAsText(objData) }}

Será renderizado como "há 1 minuto", por exemplo.

6. trim

Remove os espaços em branco no início e no final da string.

7. trimStart

Remove os espaços em branco no início da string.

8. trimEnd

Remove os espaços em branco no final da string.

9. table

Renderiza uma tabela HTML a partir de um conjunto de dados.

{{table(DataSource[0])}}
