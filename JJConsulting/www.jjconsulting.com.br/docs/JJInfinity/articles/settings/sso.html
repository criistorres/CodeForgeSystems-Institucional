Configurações SSO

Acesso via SSO

Ao ativar a opção Acesso via SSO, o usuário só poderá realizar o login utilizando o Single Sign-On (SSO). Com essa configuração, as seguintes restrições se aplicam:

  • O login somente será possível via SSO, não sendo permitido utilizar email e senha.
  • O usuário não poderá alterar a sua senha, email, configurações de autenticação de dois fatores, nem configurar ou alterar logins externos enquanto a opção de Acesso via SSO estiver ativada.

Criação Automática de Conta

Caso a opção Acesso via SSO seja habilitada e o login via SSO seja concluído com sucesso, mas o usuário não tenha uma conta no sistema Infinity, uma conta será criada automaticamente. A conta será criada sem a necessidade de mais validações, com o perfil padrão "Guest".

Esse processo simplifica o acesso de novos usuários, permitindo que eles acessem o sistema imediatamente após a autenticação via SSO.
