Configurações de Login Externo Google

Acesse os Serviços da API do Google e Serviços. Acesse sua conta Google para visualizar a pagina de APIs e serviços ativados.

Crie um projeto caso não tenha criado, selecione seu projeto para visualizar a pagina.

Vá até Tela de Permissão OAuth e Selecione seu projeto

Agora selecione o tipo Externa e depois clique no botão Criar.

Será necessario Preencher o nome do App que são exibidas na tela de consentimento.

Insira um Email para Suporte do usuário, para que entrem em contato com você a respeito do consentimento.

Adicione o Dominio que será utilizado e insira o endereço de email dos dados de contato do desenvolvedor, para notificar sobre alterações do projeto e clique em Salvar.

Avance até a Etapa Summary.

Agora clique no botão Back To Dashboard

Agora sera necessario criar credenciais, va até Credenciais.

Verifique qual projeto está selecionado e clique em Criar Crendeciais

Selecione o tipo de Credencial ID do Cliente OAuth

Escolha o tipo de aplicativo Aplicativo Web

Apos isso coloque o nome do cliente OAuth, e insira a URL (necessario https:).

Agora Clique em Criar.

Será gerado as Keys de ID Client e ID

Agora coloque as chaves nas configurações de Login Externo.
